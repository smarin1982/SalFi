**process(company_slug: str, extraction_result: ExtractionResult, country: str = "", data_dir: str = "data") -> dict:**

This is the only public function. Steps:

1. **Build one-row dict with canonical column names:**
   - `row = {"ticker": company_slug, "fiscal_year": extraction_result.fiscal_year}`
   - For each of the 22 financial columns (FINANCIALS_COLUMNS[2:]): look up `extraction_result.fields.get(canonical, np.nan)`. If not NaN, convert to USD: `to_usd(native_value, extraction_result.currency_code, extraction_result.fiscal_year)`. If NaN, keep `np.nan`.

2. **Build DataFrame and enforce schema dtypes:**
   - `df_new = pd.DataFrame([row], columns=FINANCIALS_COLUMNS)`
   - `df_new["fiscal_year"] = df_new["fiscal_year"].astype("int64")`
   - For all columns after fiscal_year: `.astype("float64")`
   - `df_new["ticker"] = df_new["ticker"].astype("object")`

3. **Determine output directory:**
   - If `country` is provided: `out_dir = Path(data_dir) / "latam" / country / company_slug`
   - Else: `out_dir = Path(data_dir) / "latam" / company_slug`
   - `out_dir.mkdir(parents=True, exist_ok=True)`

4. **Append prior-year data for growth KPI continuity (Pitfall 5 from research):**
   - `existing_path = out_dir / "financials.parquet"`
   - If `existing_path.exists()`:
     - `df_existing = pd.read_parquet(existing_path)`
     - `df_combined = pd.concat([df_existing, df_new]).drop_duplicates(subset=["fiscal_year"], keep="last")`
     - `df_combined = df_combined.sort_values("fiscal_year").reset_index(drop=True)`
     - Re-enforce all float64 dtypes on df_combined
   - Else: `df_combined = df_new`

5. **Calculate KPIs (unchanged from v1.0):**
   - `df_kpis = calculate_kpis(df_combined)` — do NOT modify this call

6. **Write Parquet atomically (unchanged save_parquet from processor.py):**
   - `save_parquet(df_combined, out_dir / "financials.parquet")`
   - `save_parquet(df_kpis, out_dir / "kpis.parquet")`

7. **Log and return summary dict:**
   - Log: `logger.info(f"latam_processor: wrote {len(df_combined)} rows for {company_slug} ({extraction_result.confidence} confidence)")`
   - Return: `{"slug": company_slug, "fiscal_year": extraction_result.fiscal_year, "confidence": extraction_result.confidence, "fields_extracted": len(extraction_result.fields), "rows_in_parquet": len(df_combined)}`
