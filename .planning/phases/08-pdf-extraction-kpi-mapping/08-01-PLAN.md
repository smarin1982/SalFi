**map_to_canonical(label: str) -> Optional[str]:**

Case-insensitive, accent-insensitive matching. Steps:
1. Lowercase the label and strip whitespace
2. Remove parenthetical content: `re.sub(r"\(.*?\)", "", normalized).strip()`
3. For each canonical key in LATAM_CONCEPT_MAP, iterate synonyms and check if `synonym in normalized or normalized in synonym`
4. Return the canonical key on first match, else `None`

Export this function as `map_to_canonical` (public name, no underscore prefix).

**parse_latam_number(text: str) -> Optional[float]:**

Handle LATAM number formatting where period=thousands separator, comma=decimal separator (e.g., `"1.234.567,89"` → `1234567.89`). Logic:
1. Strip whitespace and non-breaking spaces (`\xa0`)
2. Detect negative via `(value)` parenthesis notation
3. Strip parentheses from cleaned string
4. If both comma and period present: remove all periods, replace comma with dot
5. If only comma (no period): replace comma with dot (comma is decimal)
6. If only period: check if last segment has exactly 3 digits (thousands separator → remove period); otherwise leave as decimal separator
7. `float()` the result; return `None` on `ValueError`
8. Apply negative sign if parenthesis notation was detected

**validate_tesseract() -> bool:**

Run at module initialization (not just at OCR call time). Uses `pytesseract.get_tesseract_version()` and `pytesseract.get_languages()`. If binary not found: log error via `loguru` and return `False`. If `spa` not in languages: log warning and return `False`. Return `True` only when Tesseract is available AND `spa` language pack is present. This function must not raise — wrap in `try/except Exception`.

@dataclass
class SourceRef:
    page_number: int          # 1-indexed
    section_heading: str      # e.g. "estado de situación financiera"
    extraction_method: str    # "pdfplumber_table" | "pymupdf_text" | "ocr_tesseract"

@dataclass
class ExtractionResult:

TABLE_SETTINGS_TEXT = {"vertical_strategy": "text", "horizontal_strategy": "text", "snap_tolerance": 3}

**Single-year contract:** `extract()` extracts data for exactly one `fiscal_year` per call. When a LATAM PDF contains comparative statements (e.g., 2023 and 2022 columns side by side), the caller (`latam_processor`) is responsible for calling `extract()` twice — once per year — passing the appropriate `fiscal_year` each time. `_find_year_column()` uses the `fiscal_year` argument to select the correct column. This design keeps `ExtractionResult` as a single-year object and aligns with `latam_processor`'s concat pattern.

**extract(pdf_path: str, currency_code: str = "", fiscal_year: int = 0, country: str = "") -> ExtractionResult:**

Main entry point. The `country` parameter (e.g., `"CO"`, `"PE"`, `"CL"`) is passed through to `_score_confidence()` so that country-specific regulator critical fields can be applied. If omitted, `DEFAULT_CRITICAL_FIELDS` is used. Steps:
1. Open with `fitz.open(pdf_path)`
2. Count scanned pages using `_is_scanned_page(page)` for each page
3. If majority scanned (>50%): call `_extract_ocr(pdf_path, doc, fiscal_year, country=country)`
4. Else: call `_extract_pdfplumber(pdf_path, fiscal_year, country=country)`; if field coverage < 40% (`_fields_coverage(result) < 0.4`), attempt `_extract_pymupdf_text(doc, fiscal_year, country=country)` and use whichever returned more fields
5. Set `result.currency_code = currency_code` and `result.fiscal_year = fiscal_year` on the result
6. Return result

**_is_scanned_page(page: fitz.Page) -> bool:**
`len(page.get_text().strip()) < TEXT_CHARS_THRESHOLD and len(page.get_images()) > 0`

**_detect_section(text_lower: str) -> str:**
Check each section key in `FINANCIAL_SECTION_HEADINGS`. Return the matching section key or "unknown".

**_extract_pdfplumber(pdf_path: str, fiscal_year: int, company_slug: str = "", country: str = "") -> ExtractionResult:**

Open PDF with `pdfplumber.open()`. For each page:
1. Detect section heading via `_detect_section(page.extract_text().lower())`
2. Try `TABLE_SETTINGS_LINED`, fall back to `TABLE_SETTINGS_TEXT` if empty
3. For each table row: parse label (col 0), find numeric value with `_find_value_for_year(row, fiscal_year)`, call `map_to_canonical(label)`
4. If `map_to_canonical(label)` returns `None` (unmatched label): call `logger.debug(f"[latam_extractor] unmatched label | company={company_slug!r} | page={page_num} | label={label!r}")`. Do NOT silently skip — log then continue to next row.
5. If canonical is not None, store in fields + source_map
6. If `fiscal_year` is known, use `_find_year_column(header_row, fiscal_year)` to pick the right column; flag "fallback_year_column_used" in warnings if header-based detection fails
7. Pass `country` to `_score_confidence(fields, "pdfplumber_table", country=country)` when building the return value

**_find_value_for_year(row: list, fiscal_year: int) -> Optional[float]:**

If fiscal_year provided, look for it in a header row (detected via all-string content). If year column found, use that index. Otherwise scan right-to-left for first parseable numeric value using `parse_latam_number()`.

**_find_year_column(header_row: list, target_year: int) -> int:**

Scan header_row for a cell containing `str(target_year)`. Return column index if found. Return `1` as fallback (second column is typically most recent year in LATAM reports per research). This fallback is logged as a warning.

**_extract_pymupdf_text(doc: fitz.Document, fiscal_year: int, company_slug: str = "", country: str = "") -> ExtractionResult:**

For each page, call `page.get_text("text")`, split into lines, parse `label [whitespace] value` patterns. Use `parse_latam_number()` on the value portion. Set `extraction_method = "pymupdf_text"` in SourceRef. For each line where `map_to_canonical(label)` returns `None`: call `logger.debug(f"[latam_extractor] unmatched label | company={company_slug!r} | page={page_num} | label={label!r}")`. Do NOT silently skip. Pass `country` to `_score_confidence(fields, "pymupdf_text", country=country)` when building the return value.

**_extract_ocr(pdf_path: str, doc: fitz.Document, fiscal_year: int, company_slug: str = "", country: str = "") -> ExtractionResult:**

If `not TESSERACT_AVAILABLE`: log error and return `ExtractionResult(fields={}, source_map={}, confidence="Baja", currency_code="", fiscal_year=fiscal_year, extraction_method="ocr_unavailable", warnings=["tesseract_not_available"])`.

For each page:
1. `pix = page.get_pixmap(dpi=300)`
2. `img = Image.open(io.BytesIO(pix.tobytes("png"))).convert("L")`  (grayscale)
3. `ocr_text = pytesseract.image_to_string(img, lang="spa", config="--psm 6")`
4. Parse lines: `re.split(r"\s{2,}|\t", line)` → label + value parts
5. For each label: call `map_to_canonical(label)`. If it returns `None`: call `logger.debug(f"[latam_extractor] unmatched label | company={company_slug!r} | page={page_num} | label={label!r}")`. Do NOT silently skip.
6. If canonical is not None and value parsed successfully, store in fields + source_map. Set `extraction_method = "ocr_tesseract"` in SourceRef.
7. Pass `country` to `_score_confidence(fields, "ocr_tesseract", country=country)` when building the return value.

**_score_confidence(fields: dict, method: str, country: str = "") -> str:**

Use country-specific critical fields from `COUNTRY_CRITICAL_FIELDS` (imported from `latam_concept_map`). Fall back to `DEFAULT_CRITICAL_FIELDS` when `country` is empty or not in the dict:

```python
critical_set = COUNTRY_CRITICAL_FIELDS.get(country, DEFAULT_CRITICAL_FIELDS)
critical_found = critical_set & set(fields.keys())
n_critical = len(critical_set)
n_found = len(fields)
if n_found >= 15 and len(critical_found) == n_critical: return "Alta"
elif n_found >= 8 or len(critical_found) == n_critical: return "Media"
else: return "Baja"
```

**_fields_coverage(result: ExtractionResult) -> float:**

`len(result.fields) / 22` (22 = number of financial fields excluding ticker and fiscal_year)

**Critical constraints from research (do not violate):**
- `latam_extractor.extract()` must NOT write any files to disk — it only returns `ExtractionResult`. File writing happens in Phase 10 after human validation.

- Do not use `pdf2image` or poppler — use `fitz.Page.get_pixmap()` only.
- The TESSERACT_CMD path must be read from env var `TESSERACT_CMD` with the Windows default as fallback.
