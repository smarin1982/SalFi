# Plan 12-04 — Multi-Year Extraction from Comparative PDFs

## Objective
Extend the OCR extractor to capture both columns (current year + prior year) from comparative
financial statements — the standard format in LATAM IFRS filings that present two fiscal years
side by side. After this plan, uploading a single PDF produces two rows in financials.parquet
(e.g., 2024 and 2023), enabling YoY growth KPIs immediately.

---

## Context

### Current behaviour
- `_extract_ocr()` runs `re.finditer(r"[-]?\$?\s*\d{1,3}(?:\.\d{3})+(?:,\d{1,2})?", line)`
- Takes only the **first** match per line → captures 2024 column, discards 2023 column
- `LatamAgent` hard-codes `fiscal_year = datetime.now().year - 1` regardless of PDF content
- `ExtractionResult` is a single dataclass (one fiscal_year, one fields dict)

### Target behaviour
- Each line with two financial-format numbers → capture both as `(primary, comparative)`
- Infer actual fiscal years from PDF header text (e.g., "31 DE DICIEMBRE DEL 2024 Y 2023")
- Return `List[ExtractionResult]` with two items, one per year
- `latam_processor.process()` writes both as separate rows in financials.parquet
- `LatamAgent.run()` calls process once per result

## Tasks

### MULTI-01 — Infer fiscal years from OCR header text

In `latam_extractor.py`, add `_infer_fiscal_years(ocr_text: str) -> tuple[int, int]`:

```python
def _infer_fiscal_years(ocr_text: str) -> tuple[int, int]:
    """
    Scan first 500 chars of OCR text for a year-pair pattern like:
      "2024 Y 2023", "2024 2023", "DICIEMBRE 2024 DICIEMBRE 2023"
    Returns (primary_year, comparative_year).
    Falls back to (current_year-1, current_year-2) if not found.
    """
    import re
    from datetime import datetime
    snippet = ocr_text[:500]
    # Find all 4-digit years in range 2010-2030
    years = [int(y) for y in re.findall(r'\b(20[12]\d)\b', snippet)]
    years = sorted(set(years), reverse=True)
    if len(years) >= 2:
        return years[0], years[1]
    elif len(years) == 1:
        return years[0], years[0] - 1
    else:
        now = datetime.now().year
        return now - 1, now - 2
```
