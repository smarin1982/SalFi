# Plan 12-05 — Financial Summary Table + Working Capital Currency Fix

## Objective
Two dashboard improvements for the LATAM tab:

1. **Financial summary table** — after the KPI cards, render a compact table showing the 5
   principal financial statement lines (revenue, net_income, total_assets, total_liabilities,
   total_equity) for all loaded fiscal years. Gives analysts the raw numbers alongside ratios.

2. **Working capital in local currency** — `working_capital` is currently displayed in USD
   (it is stored in USD in kpis.parquet after FX conversion). Convert back to local currency
   when the "Moneda Original" toggle is active.

---

## Context

### Financial summary table
- `financials.parquet` holds raw values already converted to USD
- When `currency_mode == "Moneda Original"`, multiply by FX rate to show local currency
- The table must handle 1 row (one year) or 2+ rows (multi-year after Plan 12-04)
- Render with `st.dataframe()` with column formatting, not `st.table()` (no interactivity)

### Working capital currency bug
- `working_capital = current_assets - current_liabilities` — computed in `processor.py`
  from USD-denominated financials → result is in USD
- `_format_latam_kpi_value()` in `app.py` handles `format="currency_m"` type KPIs but
  working_capital uses `format="currency_b"` or similar — check `KPI_META["working_capital"]`
- The fix: in `_format_latam_kpi_value()`, when `format` is a currency type and
  `currency_mode == "Moneda Original"`, apply reverse FX conversion using
  `meta.get("currency_original")` and `meta.get("fx_rate_usd")`

## Tasks

### DASH-01 — Financial summary table in `_render_latam_tab()`

Add `_render_latam_financials_table(slug, country, currency_mode)` function:

```python
_SUMMARY_FIELDS = [
    ("revenue",           "Ingresos"),
    ("net_income",        "Utilidad Neta"),
    ("total_assets",      "Activos Totales"),
    ("total_liabilities", "Pasivos Totales"),
    ("total_equity",      "Patrimonio"),
]

def _render_latam_financials_table(slug: str, country: str, currency_mode: str) -> None:
    fin_df = st.session_state["latam_financials"].get(slug, pd.DataFrame())
    meta = st.session_state["latam_meta"].get(slug, {})
    if fin_df.empty:
        return

    available = [f for f, _ in _SUMMARY_FIELDS if f in fin_df.columns]
    if not available:
        return

    # Build display DataFrame
    display_cols = {"Año": fin_df["fiscal_year"].astype(str)}
    fx_rate = meta.get("fx_rate_usd", 1.0) or 1.0
    curr_symbol = meta.get("currency_original", "USD") if currency_mode == "Moneda Original" else "USD"
    multiplier = fx_rate if currency_mode == "Moneda Original" else 1.0

    for field, label in _SUMMARY_FIELDS:
        if field in fin_df.columns:
            display_cols[label] = (fin_df[field] * multiplier).apply(
                lambda v: f"{curr_symbol} {v/1e9:.2f}B" if abs(v) >= 1e9
                else f"{curr_symbol} {v/1e6:.1f}M" if abs(v) >= 1e6
                else f"{curr_symbol} {v:,.0f}"
                if pd.notna(v) else "N/A"
            )

    st.markdown("#### Estados Financieros")
    st.dataframe(
        pd.DataFrame(display_cols).set_index("Año"),
        use_container_width=True,
        hide_index=False,
    )
```

Call this function in `_render_latam_tab()` immediately after `_render_latam_kpi_cards()`:

```python
_render_latam_kpi_cards(active_slug, active_country, currency_mode)
_render_latam_financials_table(active_slug, active_country, currency_mode)
```

### DASH-02 — Fix working_capital to respect currency toggle

In `_format_latam_kpi_value()`, locate the branch that handles `working_capital`
(format type `"currency_m"` or `"currency_b"`) and apply FX reverse conversion:

```python
def _format_latam_kpi_value(value, fmt: str, currency_mode: str, meta: dict) -> str:
    if value is None or (isinstance(value, float) and pd.isna(value)):
        return "N/A"

    # Apply local currency conversion for monetary KPIs
    if currency_mode == "Moneda Original" and fmt in ("currency_m", "currency_b", "currency"):
        fx_rate = meta.get("fx_rate_usd", 1.0) or 1.0
        value = value * fx_rate
        curr = meta.get("currency_original", "USD")
    else:
        curr = "USD"

    if fmt == "currency_b":
        return f"{curr} {value/1e9:.2f}B"
    elif fmt == "currency_m":
        return f"{curr} {value/1e6:.1f}M"
    # ... rest of format branches unchanged
```

Check `KPI_META["working_capital"]["format"]` first to confirm the exact format key used.
