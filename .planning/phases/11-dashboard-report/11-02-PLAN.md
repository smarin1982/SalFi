    --- _load_latam_kpis(slug: str, country: str) -> pd.DataFrame ---
    Reads data/latam/{country}/{slug}/kpis.parquet.
    Returns empty DataFrame if not found.
    NOT decorated with @st.cache_data (LATAM companies change dynamically within session).

    --- _load_latam_meta(slug: str, country: str) -> dict ---
    Reads data/latam/{country}/{slug}/meta.json.
    Returns {} if not found.

    --- _load_latam_financials(slug: str, country: str) -> pd.DataFrame ---
    Reads data/latam/{country}/{slug}/financials.parquet.
    Returns empty DataFrame if not found.

    --- _format_latam_kpi_value(value: float, fmt: str, currency_mode: str, meta_info: dict) -> str ---
    If currency_mode == "USD": calls format_kpi(value, fmt) — same as S&P 500
    If currency_mode == "Moneda Original":
      - Gets fx_rate = meta_info.get("fx_rate_used", 1.0); currency_code = meta_info.get("currency_original", "USD")
      - If fx_rate and fx_rate != 0: original_value = value * fx_rate
      - Else: original_value = value
      - Formats using format_kpi(original_value, fmt) but prepends currency_code
      - e.g., "COP 4.2B" or "BRL 12.5%" — if fmt is "dollar_B", prefix currency_code
      - For non-monetary formats (percentage, ratio_x, days): values are currency-independent, return format_kpi(value, fmt) unchanged

    --- _render_latam_kpi_cards(slug: str, country: str, currency_mode: str) -> None ---
    Loads kpis.parquet and meta.json for the slug.
    Gets source_map = meta.get("source_map", {}) — maps field -> page_number.
    Renders KPI cards using the same 2/3/4/5 grid pattern as S&P 500.
    For each selected_kpis entry (reuse the sidebar's selected_kpis variable):
      - Calls st.metric(label=KPI_META[kpi]["label"], value=_format_latam_kpi_value(...))
        with delta calculation (same YoY logic as render_kpi_card)
      - Below st.metric, adds: st.caption(f"fuente: pág. {source_map.get(kpi, source_map.get('revenue', '?'))}")
        NOTE: source_map uses financial field names (revenue, net_income) not KPI names.
        Map KPI name -> financial field for source lookup:
          revenue_growth_yoy -> revenue, ebitda_margin -> revenue, net_profit_margin -> net_income,
          debt_to_ebitda -> long_term_debt, quick_ratio -> total_assets
          For all others: use "?" (no source available)
      - If more than 1 fiscal year in kpis.parquet: render trend chart via build_trend_figure()
