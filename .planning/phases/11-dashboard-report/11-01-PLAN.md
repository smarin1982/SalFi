  <name>Task 1: search_comparables() and generate_executive_report() with Claude API</name>
  <files>report_generator.py</files>
  <action>
Create report_generator.py from scratch. All three functions use lazy imports (inside the function body, wrapped in try/except ImportError) per the locked project decision.

**search_comparables(company_name: str, country: str, sector: str = "salud") -> list[dict]:**
- Lazy-imports ddgs: `from duckduckgo_search import DDGS`
- Constructs query: `f'empresas sector {sector} {country} comparables a {company_name} resultados financieros'`
- Calls `DDGS().text(query, max_results=5)`
- Returns list of dicts with keys `title`, `href`, `body` — max 3 results
- If ddgs raises any exception or returns empty: return [] (never raise — web search is optional per Phase 9 requirement SCRAP-03)

**generate_executive_report(kpis: dict, red_flags: list[dict], comparables: list[dict], company_meta: dict) -> dict:**
- Lazy-imports: `from anthropic import Anthropic` — if ImportError, return {"error": "anthropic SDK not installed"}
- Reads ANTHROPIC_API_KEY via `os.environ.get("ANTHROPIC_API_KEY")`. If absent or empty, return {"error": "[Error: ANTHROPIC_API_KEY no configurada. Agrega la clave al archivo .env]"} — do NOT raise
- Builds `system` prompt: "Eres un analista financiero senior especializado en empresas del sector salud latinoamericano. Genera reportes ejecutivos concisos y precisos en español."
- Builds `user` prompt from the four input params — include company name, country, currency, top 10 KPIs (formatted as {name}: {value}), red_flags list with severity, and comparables list. Template:
  ```
  Empresa: {company_meta['name']} ({company_meta['country']})
  Moneda base: {company_meta.get('currency', 'USD')}

  KPIs principales:
  {formatted_kpis}

  Red Flags detectadas:
  {formatted_flags}

  Empresas comparables del sector:
  {formatted_comparables}

  Genera un reporte ejecutivo con EXACTAMENTE estas cuatro secciones en JSON:
  {{
    "resumen": "Resumen de Gestión (3-4 párrafos)",
    "kpis_destacados": "KPIs Destacados (4-6 puntos clave)",
    "red_flags_activas": "Red Flags Activas (análisis de riesgos detectados)",
    "contexto_sectorial": "Contexto Sectorial (posicionamiento vs comparables)"
  }}
  Responde SOLO con JSON válido. No incluyas texto adicional fuera del JSON.
  ```
- Creates `Anthropic(api_key=os.environ.get("ANTHROPIC_API_KEY"))` client
- Calls `client.messages.create(model="claude-opus-4-6", max_tokens=4096, system=system, messages=[{"role": "user", "content": prompt}], timeout=120.0)` — 120s timeout per Pitfall 3 in RESEARCH.md
- Parses response: `msg.content[0].text` → `json.loads()`. If json.loads fails, wrap raw text in {"resumen": raw_text, "kpis_destacados": "", "red_flags_activas": "", "contexto_sectorial": ""}
- Returns the four-key dict

**Module-level imports:** only `os`, `json` — everything else is lazy inside each function.

  <action>
Add two more functions to report_generator.py:

**export_chart_png(fig) -> bytes | None:**
- Lazy-imports kaleido: `import kaleido  # noqa: F401` — if ImportError return None
- Calls `fig.to_image(format="png", width=800, height=300, scale=2)` — returns bytes
- Wraps in try/except Exception: return None on any error (kaleido Chrome not found should not crash the PDF generation flow)

**generate_pdf_bytes(report_data: dict, company_meta: dict, chart_pngs: list[bytes] | None = None) -> bytes:**
- Lazy-imports fpdf2: `from fpdf2 import FPDF` — if ImportError raise RuntimeError("fpdf2 not installed. Run: pip install fpdf2>=2.8.7")
- Creates `pdf = FPDF()`, `pdf.add_page()`, sets margins: `pdf.set_margins(20, 20, 20)`
- Font strategy: use built-in core fonts (Helvetica covers Latin-1 Spanish characters per Pitfall 6 in RESEARCH.md). Do NOT attempt Unicode TrueType font loading — Latin-1 is sufficient for Spanish.
- **Header:** company name + country + generation timestamp (formatted as "Reporte generado: {date}")
- **Four sections in order:** resumen, kpis_destacados, red_flags_activas, contexto_sectorial
  - For each section: bold section title (font size 14), then body text (font size 11, multi_cell with word wrap)
  - Use `pdf.set_font("Helvetica", "B", 14)` for titles, `pdf.set_font("Helvetica", "", 11)` for body
  - Add `pdf.ln(6)` between sections
- **Spanish character handling:** Before calling pdf.multi_cell(), encode text: `text.encode("latin-1", errors="replace").decode("latin-1")` to replace any non-Latin-1 chars with `?` rather than crashing (Pitfall 6)
- **Chart PNGs (optional):** If chart_pngs provided and non-empty, add a new page labeled "Gráficos KPI" and embed each PNG using `pdf.image(io.BytesIO(png_bytes), w=170)` with `import io` (stdlib — not lazy)
- Returns `bytes(pdf.output())`
