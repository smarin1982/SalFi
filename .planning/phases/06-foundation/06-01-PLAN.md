<feature>
  <name>FX Normalizer — currency.py</name>
  <files>currency.py, tests/test_currency.py</files>
  <behavior>
    Public API:
      to_usd(amount: float, currency: str, fiscal_year: int) -> float
        - currency="USD" → returns amount unchanged
        - currency="BRL" or "MXN" → calls _frankfurter_annual_avg(currency, year)
        - currency in {"ARS","CLP","COP","PEN"} → calls _secondary_api_rate(currency)
        - Always returns float, never None
        - Results cached: same (currency, year) call never makes a second HTTP request

      is_low_confidence_currency(currency: str) -> bool
        - "ARS" → True
        - All others ("BRL","MXN","CLP","COP","PEN","USD") → False

    Cases (test inputs → expected):
      to_usd(1000.0, "USD", 2023) → 1000.0 (identity, no API call)
      to_usd(1000.0, "BRL", 2023) → float > 0 and < 1000 (USD weaker than BRL historically)
      to_usd(1000.0, "MXN", 2023) → float > 0 and < 1000
      to_usd(1000.0, "ARS", 2023) → float > 0 (small, ARS weak vs USD)
      to_usd(1000.0, "CLP", 2023) → float > 0
      to_usd(1000.0, "COP", 2023) → float > 0
      to_usd(1000.0, "PEN", 2023) → float > 0
      is_low_confidence_currency("ARS") → True
      is_low_confidence_currency("BRL") → False
      Calling to_usd("BRL", 2023) twice → only ONE HTTP request (cache hit on second call)
  </behavior>
