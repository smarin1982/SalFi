**`_same_quarter(ts1, ts2) -> bool`** — copy VERBATIM from agent.py lines 122-136:
```python
def _same_quarter(ts1: pd.Timestamp, ts2: pd.Timestamp) -> bool:
    def _q(ts: pd.Timestamp) -> tuple:
        return (ts.year, (ts.month - 1) // 3 + 1)
    return _q(ts1) == _q(ts2)
```

**`class LatamAgent`:**

Constructor `__init__(self, name: str, country: str, url: str, data_dir: Path = DATA_DIR)`:
- `self.name = name`
- `self.country = country.upper()`
- `self.url = url`
- `self.slug = make_slug(name)`
- `self.storage_path = make_storage_path(data_dir, country, self.slug)`
- `self.meta_path = self.storage_path / "meta.json"`

`needs_update(self) -> bool`:
- If `self.meta_path` does not exist: return True
- Load meta; get `last_downloaded` str; if None/missing: return True
- `pd.Timestamp(last_dl_str)` — raises ValueError if malformed: catch and return True (log warning)
- Return `not _same_quarter(last_dl, pd.Timestamp.now())`

`run(self, force_refresh: bool = False) -> dict`:
Implement the full 6-step pipeline:
1. If `not force_refresh and not self.needs_update()`: log "Current-quarter data found — skipping scrape", call `self._process_existing(skipped_scrape=True)`, return result
2. Step 1 — Scrape: `pdf_path = latam_scraper.search_and_download(domain=self.url, slug=self.slug, storage_path=self.storage_path)`
3. Step 2 — Extract: `extraction_result = latam_extractor.extract(pdf_path)`; extraction_result has keys: "data", "confidence", "method", "source_map"
4. Step 3 — Process: `process_result = latam_processor.process(company_name=self.name, country=self.country, extracted=extraction_result["data"], storage_path=self.storage_path)`
5. Step 4 — Evaluate red flags (ONLY after Parquet files exist on disk):
   - `kpis_df = pd.read_parquet(self.storage_path / "kpis.parquet")`
   - `financials_df = pd.read_parquet(self.storage_path / "financials.parquet")`
   - `flags = evaluate_flags(kpis_df, financials_df)`
6. Step 5 — Web search enrichment (NON-BLOCKING — wrap in try/except):
   ```python
   sector_context = []
   try:
       sector_context = web_search.search_sector_context(self.name, self.country)
   except Exception as exc:
       logger.warning(f"[{self.name}] Web search failed (non-blocking): {exc!r}")
   ```
7. Step 6 — Write meta.json (only after steps 1-5 complete successfully):
   - `meta = self._build_meta(extraction_result, process_result, flags, pdf_path, scraped=True)`
   - `self._save_meta(meta)`
8. Return dict: `{"status": "success", "name": self.name, "country": self.country, "slug": self.slug, "fiscal_years": process_result["fiscal_years"], "red_flags": [vars(f) for f in flags], "sector_context": sector_context, **process_result}`

`_process_existing(self, skipped_scrape: bool = True) -> dict`:
- Called when needs_update() is False
- Read Parquet files and re-evaluate red flags (thresholds may have changed)
- Load existing meta, update red_flags_evaluated_at and red_flags_count, save
- Return same dict structure as run() but with `"status": "skipped_scrape"`

`_build_meta(self, extraction_result, process_result, flags, pdf_path, scraped) -> dict`:
Build meta.json dict with ALL fields from the research schema:
- `name`, `country`, `slug`, `url`
- `last_downloaded`: `pd.Timestamp.now().isoformat()` if scraped, else preserve existing
- `last_processed`: `pd.Timestamp.now().isoformat()`
- `fiscal_years`: `process_result.get("fiscal_years", [])`
- `fy_count`: `len(process_result.get("fiscal_years", []))`
- `status`: "success"
- `error_message`: None
- `extraction_method`: `extraction_result.get("method", "unknown")`
- `confidence`: `extraction_result.get("confidence", "Baja")`
- `approximated_fx`: `process_result.get("approximated_fx", False)`
- `ars_warning`: `self.country == "AR"` — Argentine peso devaluation warning (FX-02)
- `fields_missing`: `process_result.get("fields_missing", [])`
- `source_pdf_path`: `str(pdf_path)` if pdf_path else None
- `red_flags_evaluated_at`: `pd.Timestamp.now().isoformat()`
- `red_flags_count`: `len(flags)`

`_save_meta(self, meta: dict) -> None`:
Atomic write mirroring agent.py NTFS pattern:
```python
self.storage_path.mkdir(parents=True, exist_ok=True)
tmp = self.meta_path.with_suffix(".json.tmp")
tmp.write_text(json.dumps(meta, ensure_ascii=False, indent=2), encoding="utf-8")
if self.meta_path.exists():
    self.meta_path.unlink()  # Windows NTFS: unlink before rename
tmp.rename(self.meta_path)
```

`_load_meta(self) -> dict`:
```python
if not self.meta_path.exists():
    return {}
return json.loads(self.meta_path.read_text(encoding="utf-8"))
```
