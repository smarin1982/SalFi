    @dataclass
    class ScraperResult:
        ok: bool

        strategy: str = ""
        source_url: Optional[str] = None
        error: Optional[str] = None
        attempts: list[str] = field(default_factory=list)

        @property
        def failed(self) -> bool:

    search() — PRIMARY PATH (from RESEARCH.md Pattern 2):
    Function signature: def search(domain: str, year: int, out_dir: Path) -> ScraperResult:
    - Build 3 query variants:
      1. f'site:{domain} filetype:pdf {SEARCH_KEYWORDS_ES} {year}'
      2. f'site:{domain} filetype:pdf {SEARCH_KEYWORDS_ALT} {year}'
      3. f'site:{domain} filetype:pdf "informe anual" {year}'
    - For each query: call _ddgs_first_pdf_url(query); if pdf_url found call _download_pdf(pdf_url, out_dir, "ddgs", attempts); return immediately on success
    - Add time.sleep(random.uniform(2.0, 4.0)) between queries (not before first, not after last)
    - If no PDF URL found after all queries: return ScraperResult(ok=False, strategy="ddgs", error=f"No PDF URL found for domain={domain} year={year} after {len(queries)} queries", attempts=attempts)
    - attempts list tracks every query tried: append f"ddgs:{query[:60]}" before each attempt

    _ddgs_first_pdf_url() — implement exactly as shown in RESEARCH.md Pattern 2:
    - 3 retry attempts with exponential backoff for RatelimitException: wait = (2 ** attempt) * random.uniform(3.0, 6.0)
    - Catch TimeoutException and DDGSException: return None immediately
    - From results, return first href where href.lower().endswith(".pdf") OR ".pdf?" in href.lower()
    - (Added: check ".pdf?" in addition to endswith ".pdf" per Pitfall 1 — download scripts)

    scrape_with_playwright() — FALLBACK PATH (from RESEARCH.md Pattern 3):
    Function signature: def scrape_with_playwright(base_url: str, year: int, out_dir: Path, attempts: list) -> ScraperResult:
    - CRITICAL: Use ThreadPoolExecutor(max_workers=1) — never call _playwright_find_pdf directly
    - Submit _playwright_find_pdf(base_url, year) to executor
    - future.result(timeout=120) — catch concurrent.futures.TimeoutError → return ScraperResult(ok=False, strategy="playwright", error="Browser timeout after 120s", attempts=attempts)
    - If pdf_url returned: call _download_pdf(pdf_url, out_dir, "playwright", attempts); return result
    - If None returned: return ScraperResult(ok=False, strategy="playwright", error=f"No PDF link found on {base_url}", attempts=attempts)
    - Append f"playwright:{base_url[:60]}" to attempts before submission

    _playwright_find_pdf() — THREAD WORKER (runs inside ThreadPoolExecutor thread):
    - Creates its OWN sync_playwright() instance — not shared
    - browser = p.chromium.launch(headless=True)
    - page.goto(base_url, timeout=30_000, wait_until="domcontentloaded")
    - page.wait_for_load_state("networkidle", timeout=15_000) — catch PlaywrightTimeout, continue
    - Call _find_pdf_link_on_page(page, year) — if URL found, return it
    - If not found: iterate IR_PAGE_FRAGMENTS, try a[href*="{fragment}"] locator, click if exists, wait for networkidle, retry _find_pdf_link_on_page
    - Wrap all in try/finally: browser.close() in finally block
    - For relative hrefs (starting with /): construct absolute URL: base_parsed = from urllib.parse import urlparse; base = f"{parsed.scheme}://{parsed.netloc}"
    - Return None if nothing found

    _find_pdf_link_on_page(page, year) — scan page for PDF anchors:
    - Iterate PDF_LINK_SELECTORS
    - For each selector: links = page.locator(selector).all()
    - Prefer links where str(year) appears in href or link.inner_text()
    - Fallback: first link in selector set if no year-matched link found
    - Return href (string) or None

    _download_pdf() — SHARED DOWNLOAD (from RESEARCH.md Pattern 4):
    Function signature: def _download_pdf(url: str, out_dir: Path, strategy: str, attempts: list, timeout: int = 30) -> ScraperResult:
    - requests.head(url, timeout=timeout, allow_redirects=True)
    - Check Content-Type for "pdf" OR url.lower().endswith(".pdf") — note from research: some servers return "application/octet-stream"
    - If Content-Type clearly non-PDF (e.g., "text/html") AND url doesn't end in .pdf: return ScraperResult(ok=False, ...)
    - raw_dir = out_dir / "raw"; raw_dir.mkdir(parents=True, exist_ok=True)
    - If pdf_path.exists(): return ScraperResult(ok=True, pdf_path=pdf_path, ...) — deduplication
    - requests.get(url, stream=True, timeout=timeout); resp.raise_for_status()
    - Write chunks to pdf_path
    - After write: call _validate_pdf_magic(pdf_path); if False: pdf_path.unlink(); return ScraperResult(ok=False, error="Downloaded file is not a valid PDF (HTML interstitial suspected)")
    - Catch requests.RequestException: return ScraperResult(ok=False, error=f"Download failed: {e}", ...)

    _validate_pdf_magic(path: Path) -> bool:
    - Open in "rb" mode, read 4 bytes, return bytes == b"%PDF"

    _normalize_filename(url: str) -> str:
    - From RESEARCH.md Pattern 4 exactly: split URL, sanitize with re.sub, add .pdf extension if missing
    - If name < 8 chars or starts with "download": prefix with sha256 hash[:8]

    handle_upload() — UPLOAD PATH (from RESEARCH.md Pattern 5):
    Function signature: def handle_upload(uploaded_file, out_dir: Path) -> ScraperResult:
    - uploaded_file has .getvalue() -> bytes and .name -> str
    - raw_dir = out_dir / "raw"; raw_dir.mkdir(parents=True, exist_ok=True)
    - pdf_path = raw_dir / _normalize_filename_from_upload(uploaded_file.name)
    - pdf_path.write_bytes(uploaded_file.getvalue())
    - Return ScraperResult(ok=True, pdf_path=pdf_path, strategy="upload", source_url=None, attempts=["upload:drag-and-drop"])
    - Catch Exception: return ScraperResult(ok=False, strategy="upload", error=f"Upload save failed: {e}", attempts=["upload:drag-and-drop"])

    Define _normalize_filename_from_upload(name: str) -> str:
    - re.sub(r"[^\w.\-]", "_", name) — Windows-safe
    - If not ending in .pdf: append ".pdf"
